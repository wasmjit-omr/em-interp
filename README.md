A wasmjit-omr base tool for running WebAssembly code generated by Emscripten.

## Building

```
$ mkdir build
$ cd build
$ cmake -GNinja -DWASMJIT_OMR_LOCATION=<path to wasmjit-omr>
$ ninja
```
